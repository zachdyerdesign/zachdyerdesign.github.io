Instructions

  1. Download the files from https://github.com/zachdyerdesign/zdd
  2. Go to modules to update module data and create new modules.
  3. Use the zdd functions inside of script tags anywhere in your html to add dynamic menus and more.

Templating Functions

  1. zdd.mods.menu.get(name) - Use this to get use your dynamic menu stored in zdd.data.menus
  2. zdd.mods.include(include) - Use this to display your footer information stored in zdd.data.footer
  3. zdd.mods.icon.get(icon, size) - Use this to display an icon from font awesome. Find more icons at http://fortawesome.github.io/Font-Awesome/icons/
